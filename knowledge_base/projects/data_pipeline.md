# Real-Time Data Pipeline for E-Commerce Analytics

- **Duration**: 2023
- **Tech Stack**: Apache Spark, Kafka, Airflow, PostgreSQL, Docker, AWS S3
- **Description**: Designed and implemented an end-to-end data pipeline processing 500K+ daily transactions for an e-commerce platform.
- **Key Achievements**:
  - Built Kafka producers/consumers for real-time event streaming
  - Orchestrated ETL workflows using Apache Airflow with automated retry and alerting
  - Optimized Spark jobs reducing processing time by 60%
  - Stored processed data in PostgreSQL with dimensional modeling (star schema)
