# NLP-Based Resume Parser

- **Duration**: 2023
- **Tech Stack**: Python, spaCy, NLTK, RegEx, FastAPI
- **Description**: Developed an automated resume parsing tool that extracts structured information from PDF/DOCX resumes.
- **Key Achievements**:
  - Used Named Entity Recognition (NER) for extracting skills, experience, education
  - Built custom spaCy pipeline for domain-specific entity extraction
  - Achieved 90%+ extraction accuracy across varied resume formats
  - Exposed as REST API with FastAPI for integration
