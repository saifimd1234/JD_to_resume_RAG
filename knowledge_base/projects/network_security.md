# Network Security Anomaly Detection System

- **Duration**: 2023 – 2024
- **Tech Stack**: Python, Scikit-learn, TensorFlow, Pandas, Flask
- **Description**: Developed a machine learning-based intrusion detection system for network traffic analysis.
- **Key Achievements**:
  - Implemented multiple ML classifiers (Random Forest, SVM, Neural Networks) for attack classification
  - Achieved 97% accuracy on the NSL-KDD dataset with ensemble methods
  - Built a real-time monitoring dashboard with Flask for security alerts
  - Reduced false positive rate by 40% through feature engineering and hyperparameter tuning
