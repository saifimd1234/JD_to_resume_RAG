# Customer Churn Prediction Platform

- **Duration**: 2023
- **Tech Stack**: Python, XGBoost, Scikit-learn, Streamlit, Docker
- **Description**: Built a predictive analytics platform to identify customers at risk of churning for a telecom company.
- **Key Achievements**:
  - Engineered 50+ features from raw customer data (usage, billing, support tickets)
  - Trained XGBoost model achieving 92% AUC-ROC score
  - Deployed model via Streamlit dashboard with real-time predictions
  - Generated actionable retention strategies reducing churn by 15%
