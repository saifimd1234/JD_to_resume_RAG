# AI-Powered JD-to-Resume Generator (RAG System)

- **Duration**: 2024 – Present
- **Tech Stack**: Python, LangChain, FAISS, OpenAI, FastAPI, Streamlit
- **Description**: Built a Retrieval-Augmented Generation system that generates highly tailored resumes from job descriptions using a structured personal knowledge base.
- **Key Achievements**:
  - Designed a modular RAG pipeline with configurable chunk size, overlap, and top-K retrieval
  - Implemented ATS-friendly resume formatting with keyword optimization
  - Built a Streamlit UI with real-time resume generation, editing, and multi-format export (PDF/DOCX)
  - Achieved 85%+ ATS match scores on generated resumes
